# คู่มือการติดตั้งระบบลงทะเบียนบัตรผ่านยานพาหนะ กองบิน 23

---

## สิ่งที่ต้องมีก่อนติดตั้ง

- บัญชี Google (Gmail)
- อินเทอร์เน็ต
- เบราว์เซอร์ Google Chrome (แนะนำ)

---

## ไฟล์ในโปรเจกต์

| ไฟล์ | หน้าที่ |
|---|---|
| `code.gs` | โค้ด Google Apps Script ฝั่ง Server (บันทึกข้อมูลลง Spreadsheet) |
| `index.html` | หน้าฟอร์มลงทะเบียน |
| `css.html` | สไตล์ตกแต่งหน้าเว็บ |
| `js.html` | โค้ด JavaScript ฝั่ง Client (validation และส่งข้อมูล) |

---

## ขั้นตอนที่ 1 — สร้าง Google Spreadsheet

1. เปิด [Google Drive](https://drive.google.com) แล้วคลิก **+ ใหม่ → Google ชีต**
2. ตั้งชื่อไฟล์ เช่น `ทะเบียนบัตรผ่านยานพาหนะ บน.23`
3. ที่แถวแรก (Row 1) พิมพ์หัวคอลัมน์ดังนี้ตามลำดับ:

| A | B | C | D | E | F | G | H | I | J | K | L | M | N | O | P |
|---|---|---|---|---|---|---|---|---|---|---|---|---|---|---|---|
| ประเภทรถ | เลขทะเบียนรถ | ชื่อ สกุล (ผู้ครอบครองรถ) | เบอร์โทรติดต่อ | ชื่อ สกุล บุคคลฉุกเฉิน | เบอร์โทรบุคคลฉุกเฉิน | เลขบัตรประชาชน | ประเภทรถ (ทะเบียน) | ลักษณะรถ | จังหวัด | ยี่ห้อรถ | แบบรถ | สีรถ | วันที่เสียภาษี | วันครบกำหนดภาษี | ที่อยู่ |

4. **คัดลอก URL** ของ Spreadsheet นี้ไว้ (จะใช้ในขั้นตอนถัดไป)

---

## ขั้นตอนที่ 2 — สร้าง Google Apps Script

1. ใน Spreadsheet ที่สร้างไว้ คลิกเมนู **ส่วนขยาย → Apps Script**
2. หน้าต่าง Apps Script จะเปิดขึ้น — **ลบโค้ดเดิมทั้งหมด** ออกก่อน

---

## ขั้นตอนที่ 3 — เพิ่มไฟล์ใน Apps Script

### 3.1 แก้ไขไฟล์ `Code.gs`

วางโค้ดจากไฟล์ `code.gs` ลงในหน้าต่าง editor แล้วกด **Ctrl+S** เพื่อบันทึก

### 3.2 สร้างไฟล์ `index.html`

1. คลิก **+** ข้างคำว่า "ไฟล์" ทางซ้ายมือ → เลือก **HTML**
2. ตั้งชื่อว่า `index` (ไม่ต้องพิมพ์ `.html` — ระบบเติมให้อัตโนมัติ)
3. **ลบโค้ดเดิม** แล้ววางโค้ดจากไฟล์ `index.html` ทั้งหมด
4. กด **Ctrl+S**

### 3.3 สร้างไฟล์ `css.html`

1. คลิก **+** → **HTML** → ตั้งชื่อว่า `css`
2. ลบโค้ดเดิม วางโค้ดจากไฟล์ `css.html` ทั้งหมด
3. กด **Ctrl+S**

### 3.4 สร้างไฟล์ `js.html`

1. คลิก **+** → **HTML** → ตั้งชื่อว่า `js`
2. ลบโค้ดเดิม วางโค้ดจากไฟล์ `js.html` ทั้งหมด
3. กด **Ctrl+S**

> ⚠️ ชื่อไฟล์ต้องตรงกันพอดี (`index`, `css`, `js`) เพราะ `code.gs` เรียกใช้ชื่อเหล่านี้โดยตรง

---

## ขั้นตอนที่ 4 — Deploy เป็น Web App

1. คลิกปุ่ม **Deploy** (มุมขวาบน) → **New deployment**
2. คลิกไอคอน ⚙️ ข้าง "Select type" → เลือก **Web app**
3. ตั้งค่าดังนี้:

   | ช่อง | ค่าที่ต้องตั้ง |
   |---|---|
   | Description | `ลงทะเบียนบัตรผ่านยานพาหนะ บน.23` |
   | Execute as | **Me** (บัญชีของคุณ) |
   | Who has access | **Anyone** (ทุกคนเข้าได้ ไม่ต้อง login) |

4. คลิก **Deploy**
5. ระบบจะขอให้ **อนุญาตสิทธิ์** — คลิก **Authorize access** → เลือกบัญชี Google → คลิก **Allow**
6. หลัง deploy สำเร็จ จะได้ **Web app URL** ลักษณะนี้:
   ```
   https://script.google.com/macros/s/XXXXXXXXXX/exec
   ```
7. **คัดลอก URL นี้ไว้** — ใช้แชร์ให้ผู้ลงทะเบียน

---

## ขั้นตอนที่ 5 — ทดสอบระบบ

1. เปิด URL ที่ได้จากขั้นตอนที่ 4 ในเบราว์เซอร์
2. กรอกข้อมูลทดสอบให้ครบทุกช่อง
3. กดปุ่ม **เงื่อนไขการปฏิบัติ** อ่านเงื่อนไข แล้วปิด
4. ติ๊ก **ยอมรับเงื่อนไข**
5. กด **ลงทะเบียน**
6. ตรวจสอบใน Google Spreadsheet ว่าข้อมูลถูกบันทึกเข้าแถวใหม่

---

## การอัปเดตโค้ด (หลังแก้ไขไฟล์)

เมื่อมีการแก้ไขโค้ดในภายหลัง ต้องทำการ **Re-deploy** ทุกครั้ง:

1. คลิก **Deploy** → **Manage deployments**
2. คลิกไอคอน ✏️ ของ deployment ที่ใช้งาน
3. ช่อง Version เปลี่ยนเป็น **New version**
4. คลิก **Deploy**

> ⚠️ ถ้าแก้โค้ดแล้วไม่ Re-deploy ระบบจะยังใช้โค้ดเวอร์ชันเก่าอยู่

---

## ปัญหาที่พบบ่อย

| ปัญหา | สาเหตุ | วิธีแก้ |
|---|---|---|
| หน้าเว็บแสดงข้อผิดพลาด "Script function not found" | ชื่อไฟล์ html ไม่ตรง | ตรวจสอบชื่อไฟล์ใน Apps Script ว่าเป็น `index`, `css`, `js` |
| กด Deploy แล้วไม่ขอสิทธิ์ | เคย authorize ไปแล้ว | ปกติ ใช้งานได้เลย |
| บันทึกข้อมูลแล้วไม่ขึ้นใน Spreadsheet | Deploy ยังเป็นเวอร์ชันเก่า | Re-deploy ด้วย New version |
| เปิด URL แล้วขึ้น "You need permission" | ตั้ง access เป็น restricted | แก้ใน Manage deployments → Who has access → Anyone |
| ข้อมูลซ้ำ ลงทะเบียนไม่ได้ | เลขทะเบียนรถนั้นมีอยู่ในระบบแล้ว | ตรวจสอบใน Spreadsheet |

---

## โครงสร้างข้อมูลใน Spreadsheet

ข้อมูลจะถูกบันทึกตั้งแต่แถวที่ 2 เป็นต้นไป (แถวที่ 1 = หัวคอลัมน์)

```
Row 2 → ข้อมูลคนแรก
Row 3 → ข้อมูลคนที่สอง
...
```

ระบบจะเช็ค **เลขทะเบียนรถ (Column B)** — ถ้าซ้ำจะไม่บันทึกและแจ้งเตือน "ลงทะเบียนแล้ว"
