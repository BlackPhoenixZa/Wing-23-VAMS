# ระบบบัตรผ่านยานพาหนะ กองบิน 21 (Vehicle Permit Wing 21)

---

## 1. สรุปภาพรวมระบบ

ระบบบัตรผ่านยานพาหนะ กองบิน 21 พัฒนาบน **Google Apps Script + Google Sheets** ประกอบด้วย **4 ระบบย่อย** ที่ทำงานเชื่อมต่อกัน ดังนี้

---

### 1.1 ระบบลงทะเบียน (Registration Form)
> **ที่อยู่:** `1.ฐานข้อมูล/1.ลงทะเบียน/`

| รายละเอียด | ข้อมูล |
|---|---|
| **วัตถุประสงค์** | ให้ประชาชน/เจ้าหน้าที่กรอกข้อมูลลงทะเบียนรถยนต์/จักรยานยนต์เพื่อขอทำบัตรผ่าน |
| **ผู้ใช้งาน** | ประชาชนทั่วไป / เจ้าของรถ |
| **Google Sheet** | Sheet แรก (index 0) ของ Spreadsheet ที่ผูกกับ Script |

**ฟิลด์ข้อมูลที่เก็บ (16 ฟิลด์):**
1. ประเภทรถ (รถยนต์ / รถจักรยานยนต์)
2. เลขทะเบียนรถ *(ใช้เป็น Key ตรวจสอบซ้ำ)*
3. ชื่อ-สกุล ผู้ครอบครองรถ
4. เบอร์โทรติดต่อ
5. ชื่อ-สกุล บุคคลติดต่อฉุกเฉิน
6. เบอร์โทรบุคคลติดต่อฉุกเฉิน
7. เลขบัตรประจำตัวประชาชน
8. ประเภทรถ (ละเอียด)
9. ลักษณะรถ
10. จังหวัด
11. ยี่ห้อรถ
12. แบบรถ
13. สีรถ
14. วันที่เสียภาษีรถ
15. วันครบกำหนดเสียภาษีรถ
16. ที่อยู่

**ฟีเจอร์หลัก:**
- ตรวจสอบเลขทะเบียนซ้ำก่อนบันทึก
- แสดงเงื่อนไขการปฏิบัติ (Modal) ก่อนยืนยัน
- แสดง Popup สรุปข้อมูลหลังบันทึกสำเร็จ

---

### 1.2 ระบบฐานทะเบียน (Database Viewer)
> **ที่อยู่:** `1.ฐานข้อมูล/2.ฐานทะเบียน/`

| รายละเอียด | ข้อมูล |
|---|---|
| **วัตถุประสงค์** | แสดงรายการข้อมูลรถทั้งหมดในฐานข้อมูลเพื่อสืบค้น |
| **ผู้ใช้งาน** | เจ้าหน้าที่ / ผู้ดูแลระบบ |
| **Google Sheet** | Sheet ชื่อ `Data` ใน Spreadsheet ID: `1iH-npUr7weR9YEE0GXdhH7qPnhJ7LIGVHxmLJOmUIF0` |

**ฟีเจอร์หลัก:**
- DataTable แสดงข้อมูล 16 คอลัมน์พร้อม Responsive
- Export Excel / Copy / Print
- ค้นหาข้อมูลแบบ Real-time
- กำหนดจำนวนแถวที่แสดง (5/10/25/50/100/ทั้งหมด)

### 1.3 ระบบทะเบียนอนุมัติ (Approval Registry)
> **ที่อยู่:** `2.การขออนุมัติ/ทะเบียนอนุมัติ/`

| รายละเอียด | ข้อมูล |
|---|---|
| **วัตถุประสงค์** | บันทึกและจัดการรายการขออนุมัติบัตรผ่านยานพาหนะ |
| **ผู้ใช้งาน** | เจ้าหน้าที่ (ต้อง Login) |
| **Google Sheet** | Sheet `Data`, `setting`, `Admin`, `Status` ของ Spreadsheet ที่ผูกกับ Script |

**ฟิลด์ข้อมูลที่เก็บ (10 ฟิลด์):**
1. ลำดับ (Auto-increment ID)
2. ชื่อ-สกุล ผู้ครอบครองรถ
3. สถานะ (รออนุมัติ / ดำเนินการแล้ว)
4. เลขทะเบียนรถ
5. รายละเอียดรถ
6. ประเภทรถ
7. คิวบัตร
8. วันที่บันทึก
9. URL ภาพเอกสาร (เก็บใน Google Drive)
10. สถานะการดำเนินการ

**ฟีเจอร์หลัก:**
- **Login ด้วย Username/Password** (ตรวจสอบจาก Sheet `Admin`)
- เพิ่ม / แก้ไขรายการ พร้อมแนบภาพ
- แจ้งเตือนผ่าน **LINE Notify** เมื่อมีการบันทึก/อัปเดต
- ตัวนับสถานะ: ทั้งหมด / ดำเนินการแล้ว / รออนุมัติ
- Cache ข้อมูล 120 วินาที เพื่อลดการเรียก API

### 1.4 ระบบตรวจสอบการอนุมัติ (Approval Status Checker)
> **ที่อยู่:** `2.การขออนุมัติ/ตรวจสอบการอนุมัติ/`

| รายละเอียด | ข้อมูล |
|---|---|
| **วัตถุประสงค์** | ให้ผู้ขอตรวจสอบสถานะการอนุมัติบัตรผ่านด้วยตนเอง |
| **ผู้ใช้งาน** | ประชาชนทั่วไป / เจ้าของรถ (ไม่ต้อง Login) |
| **Google Sheet** | Sheet index 1 (ดำเนินการแล้ว), Sheet index 2 (รออนุมัติ), Sheet index 3 (ค้นหา) |

**ฟีเจอร์หลัก:**
- ค้นหาด้วย **เลขทะเบียนรถ**

- แสดง Alert เมื่อไม่พบข้อมูล

## 2. คู่มือการติดตั้ง

### สิ่งที่ต้องเตรียม
- บัญชี Google (Gmail)
- Google Spreadsheet (อย่างน้อย 4 Sheet ในแต่ละระบบ)
- Google Drive Folder สำหรับเก็บภาพ (ระบบทะเบียนอนุมัติ)
- LINE Notify Token (ถ้าต้องการแจ้งเตือน)

---

### 2.1 ระบบลงทะเบียน

**ขั้นตอนการติดตั้ง:**

1. **สร้าง Google Spreadsheet** สำหรับเก็บข้อมูลลงทะเบียน
   - กำหนดชื่อ Sheet แรกเป็นอะไรก็ได้ (ระบบใช้ `getSheets()[0]`)
   - ไม่ต้องสร้าง Header — ระบบจะเพิ่มข้อมูลต่อจากแถวสุดท้ายอัตโนมัติ

2. **เปิด Apps Script Editor**
   - ใน Spreadsheet → เมนู **Extensions → Apps Script**

3. **คัดลอกไฟล์โค้ด** ใส่ใน Apps Script Editor:
   - `code.gs` → วางในไฟล์ `Code.gs`
   - `index.html` → สร้าง HTML file ชื่อ `index`
   - `css.html` → สร้าง HTML file ชื่อ `css`
   - `js.html` → สร้าง HTML file ชื่อ `js`

4. **Deploy เป็น Web App**
   - คลิก **Deploy → New deployment**
   - Type: **Web app**
   - Execute as: **Me**
   - Who has access: **Anyone** (หรือ Anyone with Google account)
   - คลิก **Deploy** → คัดลอก URL

---

### 2.2 ระบบฐานทะเบียน

**ขั้นตอนการติดตั้ง:**

1. **เตรียม Spreadsheet ฐานข้อมูล**
   - ใช้ Spreadsheet ID: `1iH-npUr7weR9YEE0GXdhH7qPnhJ7LIGVHxmLJOmUIF0` หรือสร้างใหม่
   - ใน `code.gs` บรรทัด `var id = '...'` ให้แก้เป็น ID ของ Spreadsheet จริง
   - Sheet ต้องชื่อ **`Data`** และมี Header ที่แถวแรก

2. **สร้าง Apps Script Project ใหม่** (แยกจากระบบลงทะเบียน)
   - ไปที่ [script.google.com](https://script.google.com) → New project

3. **คัดลอกไฟล์โค้ด:**
   - `code.gs` → วางใน `Code.gs`
   - `index.html` → สร้าง HTML file ชื่อ `index`
   - `css.html` → สร้าง HTML file ชื่อ `css`
   - `js.html` → สร้าง HTML file ชื่อ `js`

4. **Deploy เป็น Web App** (ขั้นตอนเดียวกับข้อ 2.1)

> **หมายเหตุ:** ระบบนี้ดึงข้อมูลจาก Spreadsheet ที่กำหนดใน code.gs โดยตรง ต้องให้สิทธิ์เข้าถึง Drive ตอน Deploy

---

### 2.3 ระบบทะเบียนอนุมัติ

**ขั้นตอนการติดตั้ง:**

1. **สร้าง Google Spreadsheet** และสร้าง Sheet ดังนี้:

   | ชื่อ Sheet | การใช้งาน | คอลัมน์ที่ต้องมี |
   |---|---|---|
   | `Data` | เก็บรายการขออนุมัติ | A=ID, B=ชื่อ, C=สถานะ, D=ทะเบียน, E=รายละเอียด, F=ประเภทรถ, G=คิวบัตร, H=วันที่, I=URL ภาพ, J=สถานะดำเนินการ |
   | `setting` | ตั้งค่าระบบ | B2=Folder ID (Drive), B3=LINE Token, B4=TRUE/FALSE (เปิด/ปิด Notify) |
   | `Admin` | บัญชีผู้ใช้งาน | A=Username, B=Password |
   | `Status` | สรุปสถานะ (optional) | ตามที่ใช้ใน getStatus() |

2. **สร้าง Google Drive Folder** สำหรับเก็บภาพ
   - คัดลอก **Folder ID** จาก URL: `drive.google.com/drive/folders/[FOLDER_ID]`
   - วางใน Sheet `setting` เซลล์ **B2**

3. **สร้าง LINE Notify Token** (ถ้าต้องการ)
   - ไปที่ [notify-bot.line.me](https://notify-bot.line.me) → Generate token
   - วางใน Sheet `setting` เซลล์ **B3**
   - เซลล์ **B4** ใส่ `TRUE` เพื่อเปิดการแจ้งเตือน

4. **เปิด Apps Script ผ่าน Spreadsheet** → Extensions → Apps Script

5. **คัดลอกไฟล์โค้ด:**
   - `code.gs` → วางใน `Code.gs`
   - `index.html` → สร้าง HTML file ชื่อ `index`
   - `css.html` → สร้าง HTML file ชื่อ `css`
   - `form.html` → สร้าง HTML file ชื่อ `form`
   - `DataTable.html` → สร้าง HTML file ชื่อ `DataTable`
   - `js.html` → สร้าง HTML file ชื่อ `js`
   - `Login.html` → สร้าง HTML file ชื่อ `Login`

6. **แก้ไข URL ใน `code.gs`:**
   ```javascript
   tmpl.credit   = 'https://line.me/ti/p/YOUR_LINE_ID';
   tmpl.youtube  = 'https://www.youtube.com/@YOUR_CHANNEL';
   tmpl.facebook = 'https://www.facebook.com/YOUR_PAGE';
   ```

7. **กำหนดสิทธิ์ Deploy:**
   - Execute as: **Me**
   - Who has access: **Only myself** หรือ **Anyone with Google account** (แนะนำสำหรับระบบภายใน)

---

### 2.4 ระบบตรวจสอบการอนุมัติ

**ขั้นตอนการติดตั้ง:**

1. **เปิด Spreadsheet เดียวกับระบบทะเบียนอนุมัติ** แล้วเปิด Apps Script

   > หรือสร้าง Script แยก แต่ต้องแก้โค้ดให้ `openById()` ชี้ไปยัง Spreadsheet ที่ถูกต้อง

2. **โครงสร้าง Sheet ที่ต้องการ:**

   | ลำดับ Sheet | ชื่อ | เนื้อหา |
   |---|---|---|
   | Sheet index 1 | ดำเนินการแล้ว | ข้อมูลที่อนุมัติแล้ว |
   | Sheet index 2 | รออนุมัติ | ข้อมูลที่รอดำเนินการ |
   | Sheet index 3 | ค้นหา | ข้อมูลสำหรับค้นหาตามเลขทะเบียน (คอลัมน์ B = เลขทะเบียน) |

3. **คัดลอกไฟล์โค้ด:**
   - `code.gs` → วางใน `Code.gs`
   - `index.html` → สร้าง HTML file ชื่อ `index`
   - `css.html` → สร้าง HTML file ชื่อ `css`
   - `DataTable2.html` → สร้าง HTML file ชื่อ `DataTable2`

4. **Deploy เป็น Web App**
   - Who has access: **Anyone** (เพื่อให้ประชาชนเข้าถึงได้โดยไม่ต้อง Login)

### สิทธิ์ที่ต้องอนุญาตตอน Deploy ครั้งแรก

- **Google Drive** — อัปโหลดภาพ (ระบบทะเบียนอนุมัติ)
- **External URL Fetch** — ส่ง LINE Notify
