# คู่มือการติดตั้ง
## ระบบทะเบียนข้อมูลอนุมัติบัตรผ่านยานพาหนะ กองบิน 21

---

## สิ่งที่ต้องเตรียมก่อนติดตั้ง

| รายการ | รายละเอียด |
|--------|-----------|
| บัญชี Google | ต้องมีสิทธิ์ใช้ Google Drive, Google Sheets และ Google Apps Script |
| LINE Notify Token | สำหรับรับแจ้งเตือนผ่าน LINE (ถ้าต้องการ) |
| ไฟล์โปรเจกต์ | ไฟล์ทั้งหมดในโฟลเดอร์ `ทะเบียนตรวจสอบอนุมัติ` |

---

## ขั้นตอนที่ 1 — สร้าง Google Spreadsheet

### 1.1 สร้างไฟล์ใหม่
1. เปิด [Google Sheets](https://sheets.google.com) แล้วสร้าง Spreadsheet ใหม่
2. ตั้งชื่อไฟล์ว่า **"ทะเบียนข้อมูลอนุมัติบัตรผ่านยานพาหนะ"** (หรือชื่อใดก็ได้)

### 1.2 สร้าง Sheet ตามโครงสร้างนี้ให้ครบ 4 Sheet

> คลิกขวาที่แท็บ Sheet ด้านล่าง → **Insert sheet** แล้วตั้งชื่อให้ตรงตัวพิมพ์

---

#### Sheet ที่ 1 : `Data`
เก็บข้อมูลหลักของระบบ สร้าง **header row แรก** ดังนี้:

| A | B | C | D | E | F | G | H | I | J |
|---|---|---|---|---|---|---|---|---|---|
| ลำดับ | ชื่อ-สกุล | สถานะ | เลขทะเบียนรถ | รายละเอียดรถ | ประเภทรถ | คิวบัตร | วันที่ | ภาพ | สถานะอนุมัติ |

---

#### Sheet ที่ 2 : `Admin`
เก็บข้อมูลผู้ใช้งาน (username / password) สร้างข้อมูลอย่างน้อย 1 แถว:

| A (username) | B (password) | C (ชื่อแสดง) |
|---|---|---|
| admin | 1234 | ผู้ดูแลระบบ |

> ⚠️ **หมายเหตุ:** ไม่มี header row ใน sheet Admin — แถวแรกคือข้อมูล user เลย

---

#### Sheet ที่ 3 : `setting`
เก็บค่าตั้งค่าระบบ สร้างตามนี้:

| A (ชื่อ) | B (ค่า) |
|---|---|
| folder_id | *(ว่างไว้ก่อน จะกรอกในขั้นตอนที่ 2)* |
| line_token | *(ว่างไว้ก่อน จะกรอกในขั้นตอนที่ 4)* |
| line_onoff | FALSE |

> โครงสร้างนี้คือ: **row 2 col B** = Folder ID, **row 3 col B** = LINE Token, **row 4 col B** = TRUE/FALSE เปิด/ปิดการแจ้งเตือน

---

#### Sheet ที่ 4 : `Status`
ระบบใช้ข้อมูลจาก sheet นี้ในฟังก์ชัน `getStatus()` (legacy) สร้าง header แถวแรกและข้อมูลแถวสอง:

| A (ทั้งหมด) | B (ดำเนินการแล้ว) | C (รออนุมัติ) |
|---|---|---|
| ทั้งหมด | ดำเนินการแล้ว | รออนุมัติ |
| 0 | 0 | 0 |

---

## ขั้นตอนที่ 2 — สร้างโฟลเดอร์ใน Google Drive

1. เปิด [Google Drive](https://drive.google.com)
2. สร้างโฟลเดอร์ใหม่ ตั้งชื่อว่า **"vehicle-permit-images"** (หรือชื่อใดก็ได้)
3. เปิดโฟลเดอร์ที่สร้าง แล้วดู **URL ในแถบที่อยู่** จะมีรูปแบบ:
   ```
   https://drive.google.com/drive/folders/XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
   ```
4. คัดลอกส่วน ID (ตัวอักษรและตัวเลขหลัง `/folders/`)
5. นำ ID ที่ได้ไปวางใน **sheet `setting` row 2, col B**

### ตั้งค่าการแชร์โฟลเดอร์
1. คลิกขวาที่โฟลเดอร์ → **Share**
2. เปลี่ยนสิทธิ์เป็น **"Anyone with the link"** → **Viewer**
   (เพื่อให้ภาพที่อัปโหลดแสดงในระบบได้)

## ขั้นตอนที่ 3 — สร้าง Google Apps Script

### 3.1 เปิด Apps Script Editor
1. ใน Google Sheets ที่สร้างไว้ คลิกเมนู **Extensions → Apps Script**
2. Apps Script Editor จะเปิดขึ้นในแท็บใหม่

### 3.2 อัปโหลดไฟล์โปรเจกต์

ใน Apps Script Editor ให้สร้างไฟล์ตามลำดับนี้:

#### ไฟล์ที่ต้องสร้าง:

| ชื่อไฟล์ใน Editor | ประเภท | ไฟล์ต้นฉบับ |
|---|---|---|
| `Code` | Script (.gs) | `code.gs` |
| `index` | HTML | `index.html` |
| `css` | HTML | `css.html` |
| `js` | HTML | `js.html` |
| `Login` | HTML | `Login.html` |
| `DataTable` | HTML | `DataTable.html` |
| `form` | HTML | `form.html` |

#### วิธีสร้างและวางโค้ด:
1. **ไฟล์ `Code.gs`** — ระบบมีให้อยู่แล้ว ลบโค้ดเดิมออกทั้งหมด แล้วคัดลอกเนื้อหาจาก `code.gs` วางแทน
2. **ไฟล์ HTML ที่เหลือ** — คลิก **"+"** ที่แผง Files → เลือก **HTML** → ตั้งชื่อตามตาราง → วางเนื้อหาจากไฟล์ต้นฉบับ

> ⚠️ **ชื่อไฟล์ต้องตรงทุกตัวอักษร** (case-sensitive) เพราะ `code.gs` เรียกด้วย `HtmlService.createHtmlOutputFromFile('ชื่อ')`

### 3.3 แก้ไข URL ใน `code.gs`

เปิดไฟล์ `Code.gs` หาบรรทัดเหล่านี้แล้วแก้ URL ให้ตรงกับของจริง:

```javascript
tmpl.credit   = 'https://line.me/ti/p/skelparn';          // ← URL ติดต่อผู้พัฒนา
tmpl.youtube  = 'https://www.youtube.com/@wing21airbase82'; // ← YouTube channel
tmpl.facebook = 'https://www.facebook.com/wing21rtaf';     // ← Facebook page
```

## ขั้นตอนที่ 4 — ตั้งค่า LINE Notify (ถ้าต้องการ)

> ข้ามขั้นตอนนี้ได้ถ้าไม่ต้องการการแจ้งเตือน LINE

1. เปิด [LINE Notify](https://notify-bot.line.me/th/) แล้ว Login
2. คลิก **"จัดการการเชื่อมต่อ"** → **"สร้าง token"**
3. ตั้งชื่อ token เช่น `vehicle-permit-wing21`
4. เลือก Chat/Group ที่จะรับแจ้งเตือน → คลิก **"สร้าง token"**
5. คัดลอก token ที่ได้ไปวางใน **sheet `setting` row 3, col B**
6. เปลี่ยน **sheet `setting` row 4, col B** จาก `FALSE` เป็น `TRUE`

---

## ขั้นตอนที่ 5 — Deploy เป็น Web App

1. ใน Apps Script Editor คลิก **Deploy → New deployment**
2. คลิกไอคอนเฟือง ⚙️ ข้าง "Select type" → เลือก **Web app**
3. ตั้งค่าดังนี้:

| ช่อง | ค่าที่ต้องตั้ง |
|---|---|
| Description | `v1.0` (หรืออะไรก็ได้) |
| Execute as | **Me** (บัญชีของคุณ) |
| Who has access | **Anyone** |

4. คลิก **Deploy**
5. ระบบจะขอให้ **Authorize** สิทธิ์การเข้าถึง → คลิก **Authorize access** → เลือกบัญชี Google → คลิก **Allow**
6. คัดลอก **Web app URL** ที่ได้ — นี่คือลิงก์เข้าระบบ

---

## ขั้นตอนที่ 6 — ทดสอบระบบ

1. เปิด Web app URL ที่ได้จากขั้นตอนที่ 5
2. หน้า Login จะแสดงขึ้น
3. ใส่ username/password ที่ตั้งไว้ใน sheet `Admin` (ค่าเริ่มต้น: `admin` / `1234`)
4. ระบบควรโหลดและแสดงหน้าหลักได้

---

## ขั้นตอนที่ 7 — อัปเดตโค้ด (หลังแก้ไข)

เมื่อแก้ไขโค้ดแล้วต้องการให้มีผลกับระบบที่ Deploy ไปแล้ว:

1. Apps Script Editor → **Deploy → Manage deployments**
2. คลิกไอคอนแก้ไข ✏️ บน deployment ที่ต้องการ
3. เปลี่ยน Version เป็น **"New version"**
4. คลิก **Deploy**

> ⚠️ URL จะยังเป็น URL เดิม ไม่เปลี่ยน

## โครงสร้าง Spreadsheet สรุป

```
📊 Google Sheets
├── 📋 Data          ← ข้อมูลหลัก (ลำดับ, ชื่อ, สถานะ, ทะเบียน, รายละเอียด, ประเภท, คิว, วันที่, ภาพ, สถานะอนุมัติ)
├── 👤 Admin         ← ข้อมูล Login (username | password | ชื่อแสดง)
├── ⚙️ setting       ← ค่าตั้งค่า (folder_id | line_token | line_onoff)
└── 📊 Status        ← สถิติ (legacy — ทั้งหมด | ดำเนินการแล้ว | รออนุมัติ)
```

---

## โครงสร้างไฟล์ Apps Script

```
📁 Apps Script Project
├── 📄 Code.gs        ← logic หลัก (doGet, addRecord, updateRecord, getData ฯลฯ)
├── 🌐 index.html     ← หน้าหลัก (layout + include ไฟล์อื่น)
├── 🎨 css.html       ← styles ทั้งหมด
├── ⚡ js.html        ← logic ฝั่ง client (submit form, success handler)
├── 🔐 Login.html     ← login / logout / session
├── 📊 DataTable.html ← render ตาราง, edit, print
└── 📝 form.html      ← form กรอกข้อมูล
```

## การแก้ปัญหาที่พบบ่อย

| อาการ | สาเหตุ | วิธีแก้ |
|---|---|---|
| Login แล้วหน้าจอค้าง / ไม่แสดงข้อมูล | Sheet `Data` หรือ `Admin` ชื่อผิด | ตรวจสอบชื่อ sheet ให้ตรงตัวพิมพ์ |
| อัปโหลดภาพแล้ว Error | Folder ID ผิด หรือไม่ได้ตั้งค่า | ตรวจสอบค่าใน sheet `setting` row 2 col B |
| ไม่รับแจ้งเตือน LINE | Token หมดอายุ หรือ OnOff = FALSE | ตรวจสอบ token และ setting row 4 col B |
| เปิด Web App แล้วขึ้น Error "Authorization required" | ยังไม่ Authorize | Re-deploy แล้วกด Authorize ใหม่ |
| แก้โค้ดแล้วไม่มีผล | Deploy ยังเป็น version เก่า | ทำขั้นตอนที่ 7 (สร้าง New version) |
| ภาพไม่แสดงในตาราง | โฟลเดอร์ Drive ไม่ได้ตั้ง sharing | ตั้ง Anyone with the link → Viewer |
